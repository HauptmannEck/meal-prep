# Meal Prep (Meal Ops)

A generative AI-powered meal planning application designed to intelligently craft diverse, healthy, and easy-to-cook recipes based on user context, past ratings, and current cravings.

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS (Vanilla CSS in `index.html` and `index.css` for root variables and base resets)
- **Routing**: `react-router-dom`
- **Backend / BaaS**: Firebase (Firestore for database, Authentication via Google Sign-In)
- **AI Engine**: Google Gemini API (`gemini-3.5-flash`)
- **Testing**: Vitest + React Testing Library + JSDOM
- **Linting & Formatting**: Oxlint and Oxfmt (semicolons disabled)

## Features

- **Google Authentication**: Secure user login with Google via Firebase.
- **AI Meal Generator**: Uses a specialized prompt to generate 3 highly distinct recipes scaled for exactly 6 portions. Avoids recently made meals and explicitly learns from highly-rated or poorly-rated past meals.
- **Interactive Recipe Dashboard**: View generated recipes, cross off shopping list items, step through cooking procedures, and rate/review meals.
- **Robust Parsing**: Built-in JSON repair loops to handle slightly malformed Markdown or JSON generated by the AI model.

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- A Firebase Project (with Firestore and Authentication enabled)
- A Google Gemini API Key

### Installation

1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Set up your Firebase project and update `src/lib/firebase.ts` with your credentials and the Gemini API key.
4. Run `npm run dev` to start the local Vite development server.

### Available Scripts

- `npm run dev`: Start development server.
- `npm run build`: Compile TypeScript and build for production.
- `npm run preview`: Preview the production build locally.
- `npm run lint`: Run the `oxlint` linter against the codebase.
- `npm run format`: Format the codebase using `oxfmt` (semicolons disabled).
- `npm run typecheck`: Run the TypeScript compiler to verify types without emitting files.
- `npm run test`: Run the Vitest testing suite.

## Project Structure

- `src/components`: Contains core React UI components (`Dashboard.tsx`, `Generator.tsx`, `RecipeDetail.tsx`, `Header.tsx`).
- `src/lib/firebase.ts`: Firebase initialization and shared app/API keys.
- `src/types/index.ts`: TypeScript interfaces for the `Recipe` schema and other domain objects.
- `eslint.config.js` / `.oxfmtrc.json`: Configuration for code quality tools.
